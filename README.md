## Project Overview
StayHealthy is a Medical Appointment Booking website designed to help users
search for doctors, book appointments, and provide reviews based on their
consultation experience. This project focuses on creating clean and user-friendly
UI mockups for a healthcare application.

## Features
- Navigation bar with links to Home, Appointments, Health Blog, and Reviews
- User authentication screens including Sign Up and Login
- Doctor search by specialty
- Appointment booking with doctor cards
- Reviews section where patients can give feedback
- Responsive and visually appealing UI design

## Technologies Used
- HTML
- CSS
- JavaScript
- Figma (for UI mockup designs)
- Git and GitHub for version control

## Notes
- This project focuses on frontend UI design and layout.
- Backend functionality and database integration are not included.
- Screenshots of UI designs are included as part of the submission.
